This block provides access to CMS volumes stored in the local/cms plugin. 

You can use several blocks to expose distinct volumes of documentation.

see the local/cms Readme.txt for full summary of Moodle CMS embedded plugin

## Requirements

this plugin needs the local_cms plugin to be installed

## installation

Deploy the plugin into the "blocks" folder of Moodle
Run the Administration -> Notifications to complete installation

## Quick start



Add a CMS Navigation block to the Home page

Configure the block using a two step procedure :



* First create a volume and some pages in the volume

* Secondly setup again the block giving a front block title and attaching the block to a volume.
